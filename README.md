## ğŸ”§ Adding New Content

### Adding a Repository to Portfolio Index

1. Open `portfolio/index.html`
2. Find the appropriate category section
3. Add a new repository card using the template provided
4. Update the description and metadata

### Adding a New Tool

1. Create a subdirectory in `tools/`
2. Add your tool files and a README
3. Update `tools/README.md` to reference the new tool
4. Link to it from the portfolio index if it has its own repository

### Creating a Blog Post

1. Create a new file in `blog/` (organized by date or category)
2. Follow the blog post template in `blog/README.md`
3. Include metadata and proper formatting
4. Link from the main blog index page
