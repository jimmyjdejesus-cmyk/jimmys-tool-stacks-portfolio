# Pages

This directory contains static pages and documentation for the portfolio.

## Structure

Organize pages by purpose:

```
pages/
├── about/            # About pages and bio
├── skills/           # Skills and competencies
├── projects/         # Project showcases
├── contact/          # Contact information
└── resources/        # Learning resources and references
```

## How to Use

1. **Create pages** as HTML files or Markdown documents
2. **Link pages** from the main navigation
3. **Keep content updated** with current information
4. **Use consistent styling** with the main site

## Page Types

- **About**: Professional bio and background
- **Skills**: Technical skills and competencies
- **Projects**: Detailed project showcases
- **Contact**: Contact information and social links
- **Resources**: Curated learning resources

## Best Practices

- Use semantic HTML
- Ensure mobile responsiveness
- Optimize images and assets
- Include proper meta tags for SEO
- Maintain consistent navigation
