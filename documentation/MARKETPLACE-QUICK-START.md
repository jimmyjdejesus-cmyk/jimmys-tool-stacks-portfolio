## üö® Common Issues & Solutions

### **Issue: Gumroad ZIP too large**
**Solution**: 
- Use external download link (Google Drive, Dropbox)
- Or split into multiple files
- Or use Git LFS

### **Issue: Chrome extension rejected**
**Solution**:
- Check manifest v3 compliance
- Ensure privacy policy is accessible
- Fix any errors in Chrome DevTools
- Resubmit with explanation

### **Issue: Low sales**
**Solution**:
- Improve product descriptions
- Add demo videos
- Lower initial price (test)
- Add testimonials
- Share on social media
